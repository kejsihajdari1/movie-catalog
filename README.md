# Movie Catalog

A single-page web application for searching movies using the OMDb API.

## Features

- Search for movies by title
- Display top 3 search results with posters
- Click on movie poster to view detailed information

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

## Usage

1. Enter a movie title in the search bar
2. View the top 3 matching results
3. Click on any movie poster to see detailed information
4. Click outside the modal or the × button to close the details

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- OMDb API

## Build

```bash
npm run build
```
